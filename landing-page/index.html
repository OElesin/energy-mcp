Works with Home Assistant

No custom integration needed. Use HA's built-in command line sensor to pull live data.

📊 Current spot price sensor

Updates every 15 minutes with today's price stats — average, min, max, cheapest/most expensive hours.

command_line:
  - sensor:
      name: electricity_price_stats
      command: >
        curl -s -X POST https://energy-mcp.getbrechtai.com/mcp
        -H "Content-Type: application/json"
        -H "Accept: application/json, text/event-stream"
        -d '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{"name":"get_price_stats","arguments":{}}}'
        | grep "^data:" | sed 's/^data: //'
      value_template: >
        {{ (value_json.result.content[0].text | from_json).stats.avg_ct_kwh | round(2) }}
      json_attributes_path: "$.result.content[0].text"
      unit_of_measurement: "ct/kWh"
      scan_interval: 900

⚡ Cheapest EV charging window

Finds the best 3-hour window for 11 kWh. Returns the start time as state, with cost and price as attributes.

command_line:
  - sensor:
      name: cheapest_ev_window
      command: >
        curl -s -X POST https://energy-mcp.getbrechtai.com/mcp
        -H "Content-Type: application/json"
        -H "Accept: application/json, text/event-stream"
        -d '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{"name":"find_cheapest_window","arguments":{"duration_hours":3,"consumption_kwh":11}}}'
        | grep "^data:" | sed 's/^data: //'
      value_template: >
        {{ (value_json.result.content[0].text | from_json).start }}
      json_attributes_path: "$.result.content[0].text"
      scan_interval: 1800

🔌 Automation: Start charging at cheapest time

Triggers your EV charger when the cheapest window begins. Sends a notification with the cost.

automation:
  - alias: "Start EV charging at cheapest window"
    trigger:
      - platform: template
        value_template: >
          {{ now().isoformat()[:16] ==
             states('sensor.cheapest_ev_window')[:16] }}
    action:
      - service: switch.turn_on
        target:
          entity_id: switch.ev_charger
      - service: notify.mobile_app
        data:
          title: "⚡ EV charging started"
          message: >
            Charging at {{ state_attr('sensor.cheapest_ev_window',
            'avg_price_ct_kwh') }} ct/kWh.
            Estimated cost: €{{ state_attr('sensor.cheapest_ev_window',
            'total_cost_eur') }}

🌱 Carbon intensity sensor

Track how clean the grid is right now. Use to prefer green hours for flexible loads.

command_line:
  - sensor:
      name: grid_carbon_intensity
      command: >
        curl -s -X POST https://energy-mcp.getbrechtai.com/mcp
        -H "Content-Type: application/json"
        -H "Accept: application/json, text/event-stream"
        -d '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{"name":"get_carbon_intensity","arguments":{"country":"DE"}}}'
        | grep "^data:" | sed 's/^data: //'
      value_template: >
        {{ (value_json.result.content[0].text | from_json).carbon_intensity_gco2_kwh | round(0) }}
      unit_of_measurement: "gCO₂/kWh"
      scan_interval: 1800
